## 软件简介

育材堂报告助手是一款专业的材料试验数据处理和报告生成桌面工具软件。本软件集成Origin绘图引擎，可自动化处理拉伸试验、VDA弯曲试验、硬度测试等多种材料力学性能测试数据，并自动生成规范的PPT报告文档。

## 功能模块

### 1. 拉伸报告
- 支持 Word/Excel 数据导入
- 自动提取试样编号、厚度、Rp、Rm、Ag、A 等参数
- 自动计算平均值±标准差
- 一键生成 PPT 报告

**Origin绘图功能：**
- 自动识别曲线数据表
- 自动提取试样编号并替换Y轴列名（如"应力01"→"试样编号"）
- 支持XY列调换（原始数据为YX格式时自动转换为XY）
- 自动设置XYXY列类型（X列设为X轴，Y列设为Y轴）
- 支持自定义每图曲线数（默认12条）
- 支持Origin模板选择
- 支持自定义图片尺寸（默认15cm×12cm）
- 自动复制为OLE对象，可在PPT中双击编辑

### 2. VDA弯曲
- 支持 Excel/CSV 数据导入
- 自动提取试样编号、厚度、最大力、压头位移、角度等参数
- 单位自动转换（N→kN）
- 一键生成 PPT 报告

**Origin绘图功能：**
- 自动识别原始数据表（包含"原始数据"或"VDA"的工作表）
- 自动提取试样编号并替换Y轴列名（如"力_1"→"试样编号"）
- 支持XY列调换
- 自动设置XYXY列类型
- 支持自定义每图曲线数（默认12条）
- 支持Origin模板选择
- 支持自定义图片尺寸（默认15cm×12cm）
- 使用Ctrl+J复制为OLE对象

### 3. 硬度提取
- 批量处理硬度数据

### 4. 相变点绘图
- 支持拖拽多个 CSV 文件
- 智能识别CSV表头，自动匹配温度列(Temperature)和长度变化列(Change in Length)
- 支持欧洲CSV格式（分号分隔符，逗号作小数点）
- 每个文件生成一张图
- 支持 Origin 模板选择
- 支持自定义图片尺寸（默认11cm×8.8cm）
- 自动导出为 PPT（每页一张OLE图形）

## Origin绘图核心特性

| 特性 | 说明 |
|------|------|
| XY列调换 | 原始数据为YX格式时，自动调换为XY格式 |
| XYXY列类型 | 自动将偶数列设为X轴(type=4)，奇数列设为Y轴(type=1) |
| 试样编号替换 | 从汇总表提取试样编号，替换Y轴列名 |
| OLE对象 | 使用Ctrl+J复制，可在PPT中双击编辑Origin图形 |
| 剪贴板检测 | 自动检测复制是否成功，失败时自动重试 |
| 图片尺寸 | 支持自定义宽度和高度（单位：cm） |

## 界面特性

- 默认亮色主题，支持深色/亮色切换
- 拖拽文件支持
- 绘图前警告提示（请勿操作键盘鼠标）
- 完成后自动打开生成的PPT
